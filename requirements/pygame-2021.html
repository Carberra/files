game development with pygame (2021)

Required

  • Python 3.5.0 or greater*
  • pygame 2.0.0 or greater

Optional

  • numpy 1.6.2 or greater

Notes

*Pygame recommends Python 3.7.7 or greater for performance reasons.

**This is only required if you want to use the SurfArray module. It is not used in the series.
