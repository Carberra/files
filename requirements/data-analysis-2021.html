data analysis and visualisation in python (2021)

Required

  • Python 3.7.1 or greater
  • A Jupyter notebook compatible editor, such as Visual Studio Code or the Jupyter online editor
  • pandas 1.2.0 or greater
  • seaborn 0.11.0 or greater
  • scikit-learn 0.20.0 or greater
  • openpyxl 3.0.0 or greater
  • ipykernel 5.0.0 or greater
  • Some data to analyse! If you're stuck for ideas, you can use these datasets

Optional

  • analytix 1.1.1 or greater*

Notes

*This library was used to get the data being analysed, and does not feature in the series.
