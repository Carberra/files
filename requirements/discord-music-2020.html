building a discord.py music bot (2020)

This series first launched on 19 Aug 2020. It will be supported until 1 Apr 2022. What does this mean?

Required

  • Python 3.5.0 or greater
  • A Discord application
  • OpenJDK 13.0.2
  • Lavalink*
  • The configuration file
  • discord.py[voice] 1.5.0 or greater**
  • wavelink 0.9.0 or greater

Notes

*Lavalink 1079 no longer works. You should use the latest version instead.

**Some episodes do not use this version, but breaking changes mean you can no longer use versions prior to 1.5.0.
