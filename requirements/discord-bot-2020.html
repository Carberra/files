building a discord.py bot (2020)

Required

  • Python 3.5.0 or greater
  • A Discord application
  • discord.py 1.5.0 or greater*
  • discord-ext-menus 1.0.0a21 or greater**
  • apscheduler 3.0.0 or greater
  • better-profanity 0.6.0 or greater
  • psutil 5.8.0 or greater

Optional

  • DB Browser for SQLite
  • FileZilla

Notes

*Some episodes do not use this version, but breaking changes mean you can no longer use versions prior to 1.5.0.

**You cannot install this directly from PyPI. Consult the introduction video for more information.
